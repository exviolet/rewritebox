# Features Backlog

Фичи реализованные после Phase 6. Все завершены.

---

## 1. Diff-превью перед Apply Preset ✅

Перед применением пресета показать что именно изменится: количество замен, затронутые фрагменты. Пользователь подтверждает или отменяет.

- [x] Функция `previewReplacePairs()` в `replaceEngine.ts` — возвращает список замен с контекстом
- [x] UI превью в `PresetsPanel` — список "было → стало" перед Apply
- [x] Кнопки "Применить" / "Отмена"

## 2. Импорт/экспорт пресетов ✅

Шаринг пресетов через `.json` файлы между устройствами или коллегами.

- [x] Экспорт: скачать выбранный пресет как `.json`
- [x] Импорт: загрузить `.json` и добавить пресет в список
- [x] Валидация структуры при импорте

## 3. Тёмная/светлая тема ✅

Переключение темы. Настройка хранится в Settings / IndexedDB.

- [x] CSS-переменные для светлой темы в `index.css`
- [x] Toggle в хедере (солнце/луна) + в Settings панели
- [x] Персистенция выбора в IndexedDB (`meta` store)

## 4. Markdown preview + сохранение в .md ✅

Поддержка Markdown: превью и сохранение не только в `.txt`.

- [x] Toggle для Markdown-превью (Ctrl+M)
- [x] Рендеринг через `marked` v17
- [x] Download dropdown с выбором формата (.txt / .md)

## 5. Settings панель ✅

Панель настроек с базовыми параметрами редактора.

- [x] Размер шрифта (slider + кнопки ±, 10–24px, default 13px)
- [x] Перенос строк (wrap/no-wrap toggle)
- [x] Тема (дублирование toggle из хедера)
- [x] Персистенция в IndexedDB (`meta` store, нативный boolean)
- [x] Шорткат Ctrl+, и кнопка-шестерёнка в хедере
- [x] Мутуальная исключительность через единый `SidePanel` state
