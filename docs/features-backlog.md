# Features Backlog

Фичи запланированные после Phase 6, до Tauri-обёртки (Phase 7).

---

## 1. Diff-превью перед Apply Preset

Перед применением пресета показать что именно изменится: количество замен, затронутые фрагменты. Пользователь подтверждает или отменяет.

- [ ] Функция `previewReplacePairs()` в `replaceEngine.ts` — возвращает список замен с контекстом
- [ ] UI превью в `PresetsPanel` — список "было → стало" перед Apply
- [ ] Кнопки "Применить" / "Отмена"

## 2. Импорт/экспорт пресетов

Шаринг пресетов через `.json` файлы между устройствами или коллегами.

- [ ] Экспорт: скачать выбранный пресет как `.json`
- [ ] Импорт: загрузить `.json` и добавить пресет в список
- [ ] Валидация структуры при импорте

## 3. Тёмная/светлая тема

Переключение темы. Настройка хранится в Settings / IndexedDB.

- [ ] CSS-переменные для светлой темы в `index.css`
- [ ] Toggle в Settings
- [ ] Персистенция выбора в IndexedDB (`meta` store)

## 4. Markdown preview + сохранение в .md

Поддержка Markdown: превью и сохранение не только в `.txt`.

- [ ] Split-view или toggle для Markdown-превью
- [ ] Рендеринг MD (библиотека или свой парсер — TBD)
- [ ] Ctrl+S — выбор формата (.txt / .md) или автоопределение по расширению таба
- [ ] Download кнопка поддерживает оба формата

## 5. Settings

Панель настроек с базовыми параметрами редактора.

- [ ] Размер шрифта (slider или ±, default 13px)
- [ ] Тема (тёмная/светлая) — см. пункт 3
- [ ] Перенос строк (wrap/no-wrap)
- [ ] Персистенция настроек в IndexedDB
- [ ] Шорткат или кнопка для открытия Settings
